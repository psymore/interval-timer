What it does

Small, focused, and it stays out of your way

Alarms that don't sound like alarms

Point the alarm at a local sound file, a YouTube video, or a Spotify track. It falls back to a local sound automatically if a source is unreachable.

Keeps ticking, even minimized

Built on an elapsed-time loop rather than tick-counting, so it self-corrects instead of drifting or freezing when the window is backgrounded.

Presets for how you actually work

Save your usual work/break/loop combinations and switch between them from a dropdown instead of re-typing numbers every session.

An always-on-top mini window

Pop out a small frameless timer that floats above everything else while you work, synced live with the main window.
